# hello-idea-gradle

Example of an IntelliJ IDEA Gradle project with support for Kotlin, modified to
build a fat jar.

To create such a project:
 - File -> New -> Project
 - Gradle, +Kotlin
 - Rest: defaults can be used.
 - After the project is created, modify `build.gradle`. Update the `jar`
   plugin configuration to follow this project's configuration.
